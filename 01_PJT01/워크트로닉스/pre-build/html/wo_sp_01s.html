국·내외를 아우르는 차세대 RPA로써 업무 효율성을 극대화할 수 있는 솔루션입니다.

프로세스마이닝 / 저장 및 분석

  • 웹 기반의
    프로세스
    설계

    1. Work Designer

    드래그 앤 드랍 방식의 플로우 차트를 이용하여 프로세스의 흐름을 직관적으로 파악하며 설계할 수 있는 개발환경제공

  • 스케쥴링 /
    모니터링 /
    프로세스 관리

    2. Work Manager

    멀티 봇 관리, 봇 스케줄링, 모니터링 및 봇 별 로그 저장등 웹 기반의 통합 관리를 수행하며 실시간으로 봇에 대한 로그 수집 및 분석 기능 제공

  • 프로세스
    구현 및
    데이터 저장

    3. Work Bot

    웍 디자이너에서 설계된 프로세스를 빠르고 안정적으로 수행하여 봇의 가용성을 증가하도록 설계됨

  • 4. 문자 인식 및 영상 인식(OCR)

    다큐먼트 및 이미지에 문자를 인식하여 봇과 협업하여 업무를 자동화 할 수 있도록 함

  • 5. Chatbot - 대화형

    자연어 처리 및 인식을 통하여 대화형으로 봇과 통신하여 프로세스를 자동화 할 수 있음

  • 6. Rule Engine Based on AI

    봇 실행시의 모든 인스턴스 로그 및 웹 스크래핑을 통해 데이터를 저장하고 분석을 통하여 인사이트를 얻음

자주 묻는 질문을 확인해보세요

웍트로닉스는 산업 / 직무에 상관없이 사용 가능한가요?

예. 웍트로닉스는 규칙성이 있는 정형화된 업무부터 다소 복잡한 프로세스를 가진 업무까지 자동화를 실현할 수 있습니다.

웍트로닉스는 어떻게 사용 가능한가요?

웍트로닉스는 웹 기반 RPA 솔루션으로 PC를 통해 Chrome, Edge등 웹 브라우저를 통해 접속하셔서 계정 생성하여 바로 사용하실 수 있습니다. (*참고: 웍트로닉스는 유료로 라이선스를 구매하여 사용 가능합니다.

웍트로닉스는 자체 개발 솔루션 인가요?

예. 웍트로닉스는 이든티앤에스에서 자체 개발한 솔루션입니다.

웍트로닉스는 개발자가 아니더라도 사용할 수 있나요?

웍트로닉스는 로우코드를 기본으로 하여 개발자가 아니더라도 손쉽게 라이브러리 등을 활용해 자동화 프로세스를 설계 할 수 있습니다. 웍 디자이너(Work Designer)에서 Drag & Drop으로 자동화 스크립트를 순서대로 생성합니다.
