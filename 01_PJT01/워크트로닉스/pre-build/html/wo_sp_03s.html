웍디자이너

설계, 검증이 가능한 Studio

웍디자이너는 업무 자동화 스크립트를 설계할 뿐만 아니라 자동화 스크립트 라이브러리를 제공하는 Studio입니다.

장벽 없는 손쉬운 개발환경

가로형 플로우차트 기반의 Drag&Drop 방식으로 개발자는 물론 현업 사용자도 웍트로닉스를 이용하여 본인의 업무를 손쉽게 자동화 하는 것이 가능합니다.

클라우드 기반으로 IT 자원 활용 최적화

PC에 RPA Bot만 설치하고 프로세스 설계는 웹을 통해 클라우드 상에서 함으로서 IT 자원 활용을 최적화 할 수 있습니다.

정말 어렵지 않아요! PLAN · DO · SEE만 기억하면 됩니다.

PLAN

웍 디자이너 프로세스 작성

  • Flow Chart Drag & Drop 방식으로 사용자 친화적인 인터페이스 제공

  • 사용자 등록 라이브러리를 통한 Activity 재사용

  • 재활용 함수 등록을 통한 Activity 재사용

  • 생성 프로세스의 확장성(수정/변경 및 복사 기능)

  • 라이브러리, Activity 등 검색기능

  • Script 모드와 병행 가능

레코딩 기능

  • Windows Application 자동화

  • Web Application 자동화 기능

  • 레코딩을 통한 업무 프로세스의 전체적인 아키텍처 구현 기능

  • UI 및 오브젝트 인식 기능

  • XPath기반의 웹 Element 자동 인식 기능

  • 사용자의 마우스 키보드 입력을 인식하여 Script로 변환

DO

워크플로우 모니터링 / 스케줄링

로봇 관리 및 모니터링 기능

  • 로봇 운용 현황 실시간 모니터링 기능

  • 제별, 로봇 PC별 수행 업무 목록/내용/결과, 작업 현황, 에러 현황 모니터링)

  • 프로세스 배포 및 수행 명령

  • Project 그룹 생성 및 제거 등

스케줄링

  • Bot & Work Designer만으로 스케쥴링 기능 실행

  • 특정날짜 / 요일 기준 반복 실행 설정, Triggering

디버깅 모드

  • Flow Chart 기반 RPA 개발 솔루션을 통해 개발 Console 오류 메시지 표시, Debugging 기능 지원

  • 디버그 추적 기능 : 에러 발생 단계 / 시간 / 원인 확인 가능

  • Flow Chart 상에서 Script 와 동일한 수준의 오류 핸들링 기능 지원

SEE

대시보드 및 통계 기능

  • 상세한 로봇 운영 결과 및 주요지표(KPI) 관리 기능 지원

  • 기간, 단위 업무, 사용자별 로봇 관련 통계

  • Bot 명세(부서별, 업무별, PC별) 및 자원 사용량 모니터링 기능

  • Work Manager 도입 없이, Work Designer상에서 기본 Bot 및 과제수행 모니터링 기능 제공

로그 관리

  • 중앙 관리 서버 및 로봇 PC 내 다양한 로그 정보 관리

  • 각종 로그 및 오류 명세 등 기록 및 알림

  • 감사/실행 로그 관리 기능

알림 기능

  • RPA 작업에 대한 진행상황을 이메일/아웃룩/메신저로 전달

  • RPA 작업 도중 예기치 않은 상황에 대한 보고

  • Desktop에서 발생할 수 있는 다양한 오류로 인해 자동화 작업 중 일부 미수행 또는 오류가 발생할 경우 사용자에게 알림으로서 업무 중 사고 발생을 미연에 방지
